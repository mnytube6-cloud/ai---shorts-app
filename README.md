# MNYTube Cloud

This is a SaaS app to convert long YouTube videos into Shorts automatically.

## Features
- Upload YouTube video link
- Auto cut into Shorts
- AI captions
- Font & color styles
- Payment system (INR)
- Subscription & credit system

## Tech Stack
- Frontend: React
- Backend: Node.js
- AI: Python
- Database: MongoDB
